
# giant-clam-website

# Setup

1. clone the git
```
git clone https://github.com/A2tech-Sdn-Bhd/giant-clam-website.git
```

2. change directory to the git folder
```
cd giant-clam-website
```

3. change branch to dev
```
git checkout dev
```

4. install all depedency
```
npm install
```

5. run the website
```
npm run dev
```
